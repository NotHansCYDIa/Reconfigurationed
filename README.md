<h1 align="center"> Rechecked </h1>

<p align="center">
  A custom terminal that runs In C# and was made in VSC on a macOS. <br>

- [x] import command
- [x] clear command
- [x] reset command
- [ ] echo command
- [ ] access command
- [ ] system command
- [ ] -info extension
- [ ] -local extension
- [ ] -properties extension
- [ ] -config extension
